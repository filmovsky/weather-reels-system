\# Weather Reels System



Automatyczny system do generowania i publikowania krótkich rolek z prognozą pogody.



\## Funkcje systemu



\- pobieranie danych pogodowych

\- analiza prognozy

\- generowanie tekstu prognozy

\- generowanie voice-over

\- renderowanie wideo 9:16

\- automatyczna publikacja na platformach short-video



\## Platformy publikacji



\- TikTok

\- YouTube Shorts

\- Instagram Reels

\- Facebook Reels



\## Główne technologie



\- Python

\- Open-Meteo API

\- ElevenLabs TTS

\- FFmpeg

\- Redis

\- Celery



\## Struktura projektu



\- app — logika systemu

\- config — konfiguracja systemu

\- assets — zasoby graficzne i audio

\- storage — wygenerowane dane

\- scripts — skrypty uruchamiające
